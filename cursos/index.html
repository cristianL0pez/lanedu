Desarrollo con Django e IA aplicada

Intermedio / Avanzado

Construcción de aplicaciones web con Django integrando modelos de IA (LLMs, APIs de IA, asistentes y chatbots), buenas prácticas de arquitectura y despliegue básico.

  • Duración sugerida: 60–90 horas
  • Requisitos: Python básico, HTML/CSS básico
  • Enfoque: proyectos web + integración de IA

Docker desde cero para desarrolladores y DevOps

Intermedio

Fundamentos de contenedores, Docker CLI, imágenes, volúmenes, redes y Docker Compose. Preparado para que los estudiantes trabajen con proyectos reales.

  • Duración sugerida: 30–60 horas
  • Requisitos: Conocimientos básicos de Linux o línea de comandos
  • Enfoque: laboratorios prácticos con contenedores

Linux para infraestructura y DevOps

Inicial / Intermedio

Introducción a Linux enfocada en administración de servidores, gestión de usuarios, permisos, servicios, networking y automatización básica con scripts.

  • Duración sugerida: 40–60 horas
  • Requisitos: Manejo básico de PC
  • Enfoque: administración y operación de servidores

CI/CD y automatización de pipelines

Intermedio

Fundamentos de Integración Continua y Entrega Continua utilizando herramientas como Azure DevOps y/o GitHub Actions, pruebas automatizadas, despliegue a entornos y manejo de secretos.

  • Duración sugerida: 30–50 horas
  • Requisitos: Conocimientos de git y algún lenguaje
  • Enfoque: pipelines reales sobre proyectos de ejemplo
Cotizar este curso

Seguridad básica y hardening de servidores

Seguridad

Conceptos esenciales de ciberseguridad, amenazas habituales, hardening de sistemas Linux, manejo de logs y buenas prácticas para entornos on-premise y en la nube.

  • Duración sugerida: 30–40 horas
  • Requisitos: Linux intermedio recomendado
  • Enfoque: prácticas reales en laboratorios aislados

Virtualización y laboratorios con Proxmox

Infraestructura

Uso de Proxmox para crear laboratorios de máquinas virtuales, redes internas, snapshots y escenarios de práctica para cursos de TI, DevOps y seguridad.

Modalidades de dictación

Todos los cursos se pueden adaptar a modalidad online sincrónica, híbrida o presencial, con laboratorios en la nube o en infraestructura local.

Online sincrónico

Clases en vivo por videoconferencia, acceso a laboratorios remotos y acompañamiento al relator durante las sesiones clave.

Híbrido

Combinación de sesiones en vivo y trabajo autónomo en los laboratorios, ideal para programas largos o de nivelación.

Presencial con laboratorios remotos

Los estudiantes están en sala, pero los entornos corren en la infraestructura de LanEdu, evitando depender de PCs locales.
